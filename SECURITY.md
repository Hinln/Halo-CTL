# 安全策略 / Security Policy

## 支持范围 / Supported Versions

仅支持最新的 `main` 分支以及最近一个已发布的 tag 版本。

Only the latest `main` branch and the most recent tagged release are supported.

## 漏洞上报 / Reporting a Vulnerability

请不要在公开 Issue 中报告安全问题。

Please do not open public issues for security reports.

- 邮箱 / Email：security@your-domain.example（请替换为你的邮箱 / replace this）
- 请附带 / Include：受影响版本、复现步骤、影响评估、以及必要的日志/截图（注意去除密钥）

We will respond as soon as possible.
