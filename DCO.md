# 贡献者协议（DCO）/ Developer Certificate of Origin

本项目采用 DCO（Developer Certificate of Origin）作为贡献者协议。

当你提交代码时，表示你同意并声明：你有权提交该代码，或该代码来源与许可满足开源贡献要求。

建议在每次提交信息中附加一行：

```
Signed-off-by: Your Name <your.email@example.com>
```

你可以通过以下方式快速签署：

```bash
git commit -s
```

该模式不要求签署独立 CLA，但能在开源协作中提供基本的版权归属声明。

参考：

- https://developercertificate.org/
